## 조건문

### if / else if / else

> 조건이 **참(true)**일 때 블록을 실행한다. 조건은 **소괄호 `()`**, 블록은 **중괄호 `{}`**로 감싼다. (파이썬은 콜론+들여쓰기)

```cpp
int score = 85;

if (score >= 90) {
    std::cout << "A\n";
} else if (score >= 80) {
    std::cout << "B\n";     // 실행됨
} else {
    std::cout << "C\n";
}
```

| C++ | 파이썬 대응 |
|-----|-------------|
| `if (...)` | `if ...:` |
| `else if` | `elif` |
| `else` | `else` |

> ⚠️ **`=`와 `==` 혼동 주의.** `if (x = 5)`는 비교가 아니라 **대입**이라 항상 참이 된다. 비교는 반드시 `==`.
> ```cpp
> if (x == 5) { }   // ✅ 비교
> if (x = 5)  { }   // ❌ 대입 → 항상 참 (단골 버그)
> ```

> 💡 C++ 조건은 **0이면 거짓, 0이 아니면 참**이다. `if (n)`은 `if (n != 0)`과 같다.

### switch

> 하나의 값을 **여러 경우(case)로 분기**한다. 파이썬엔 없던 문법 (3.10의 `match`와 유사).

```cpp
int day = 3;
switch (day) {
    case 1: std::cout << "월\n"; break;
    case 2: std::cout << "화\n"; break;
    case 3: std::cout << "수\n"; break;   // 실행됨
    default: std::cout << "기타\n";
}
```

> ⚠️ **`break`를 빠뜨리면 다음 case로 줄줄이 실행된다(fall-through).** switch 최대 실수.
> ```cpp
> switch (x) {
>     case 1: doA();          // break 없음 → case 2도 실행!
>     case 2: doB(); break;
> }
> ```
> (반대로 여러 case를 묶고 싶을 땐 fall-through를 일부러 이용하기도 한다.)

## 단순 반복문

### for

> C++ `for`는 `(초기화; 조건; 증감)` 세 부분을 직접 제어한다. (파이썬 `for x in range()`와 형태가 다름)

```cpp
for (int i = 0; i < 5; i++) {
    std::cout << i << " ";   // 0 1 2 3 4
}
```

| | 파이썬 | C++ |
|--|--------|-----|
| 형태 | `for i in range(5):` | `for (int i = 0; i < 5; i++)` |
| 제어 | range가 알아서 | 초기화·조건·증감 직접 |

### while / do-while

```cpp
int n = 0;
while (n < 3) {        // 조건 먼저 검사
    std::cout << n;    // 0 1 2
    n++;
}

int m = 0;
do {
    std::cout << m;    // 0 1 2 (최소 1번은 실행)
    m++;
} while (m < 3);
```

> 💡 `do-while`은 **조건 검사 전에 일단 한 번 실행**한다. 파이썬엔 없는 형태라, "최소 1회 실행"이 필요할 때 쓴다.

### break / continue

> 파이썬과 동일. `break` 즉시 탈출, `continue` 다음 반복으로.

```cpp
for (int i = 0; i < 10; i++) {
    if (i == 5) break;          // 0~4까지만
    if (i % 2 == 0) continue;   // 짝수 건너뜀
    std::cout << i << " ";      // 1 3
}
```

## 다중 반복문

### 중첩 반복문

> 반복문 안의 반복문. 2차원 데이터 처리에 필수.

```cpp
for (int i = 0; i < 3; i++) {
    for (int j = 0; j < 3; j++) {
        std::cout << i << "," << j << "  ";
    }
    std::cout << "\n";
}
```

### 2차원 순회 패턴

> 격자(2차원 배열) 문제의 기본 틀. 코딩테스트 단골.

```cpp
int grid[3][3];
for (int r = 0; r < 3; r++) {        // 행(row)
    for (int c = 0; c < 3; c++) {    // 열(col)
        grid[r][c] = r * 3 + c;
    }
}
```

> 💡 중첩 안에서 `break`는 **안쪽 반복문만** 빠져나간다. 바깥까지 한 번에 나가려면 플래그 변수나 함수 분리를 쓴다. (C++엔 `goto`도 있지만 지양)

## 참고

### 삼항 연산자

> `조건 ? 참값 : 거짓값` — 한 줄 분기. (파이썬 `참값 if 조건 else 거짓값`에 대응)

```cpp
int age = 20;
std::string result = (age >= 18) ? "성인" : "미성년자";
std::cout << result;   // 성인
```

### 무한 루프

> 조건을 항상 참으로 두는 반복. `break`로 탈출.

```cpp
while (true) {
    // ...
    if (조건) break;
}

for (;;) {       // 같은 의미 (조건 생략 = 무한)
    // ...
}
```

> ⚠️ 탈출 조건·증감을 빠뜨리면 프로그램이 멈추지 않는다. 무한 루프엔 반드시 `break` 경로를 둬라.
